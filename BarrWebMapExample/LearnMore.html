Using map services published via ArcGIS Server, Barr is able to provide access to GIS data and web-based maps via the Internet. Web maps provide a smart, intuitive framework for reviewing, editing, and interacting with the data. These maps can typically be accessed via desktop, laptop, or tablet computers as well as with smart phones.

Barr develops web maps for a variety of purposes, although the most common is to allow immediate access to project data by the project team and client. Web maps act as a secure, central “storage area” for data to provide easy access to the most up-to-date information, including monitoring data, photographs, project design and layout, and project impacts. The web maps also allow multiple users to add and mark-up data stored in a secured relational database as well as edit the same features at the same time.

The web maps also allows multiple users to add and mark-up data stored in a secured relational database located on Barr's GIS server. In this environment, multiple users can edit the same features at the same time. Follow the steps outlined below to edit data in the Barr Web Map Example application:

1. Click on the "Editor" heading in the left pane
2. Click the "Start Editing" button
3. Select the type of Comment from the menu and then click or tap on the map where at the location where the comment should get placed
4. Populate the pop-up attribute window (note: an attachment can be added to the comment by browsing to the file)
5. Use tools located on the bottom of the Editing tools as needed
6. Click the "Stop Editing" button when edits are complete
