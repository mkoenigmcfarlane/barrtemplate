Map navigation using mouse and keyboard:
  • Current Position: Click on "Track Location" button in upper left corner of the map to zoom to current location. Must enable or allow location services
  • Drag to pan
  • SHIFT + Click to recenter
  • SHIFT + Drag to zoom in
  • SHIFT + CTRL + Drag to zoom out
  • Mouse Scroll Forward to zoom in
  • Mouse Scroll Backward to zoom out
  • Use Arrow keys to pan
  • + key to zoom in a level
  • - key to zoom out a level
  • Double Click to Center and Zoom in
  • Layers: Controls map layer visibility and displays map symbols/legend.
  • Bookmarks: Create bookmarks to save a defined map extent. User bookmarks will persist in the map until the web browser cache is cleared.
    • Draw: Add graphics to the map. Graphics will show up on a printed copy of the map, but all graphics will get cleared from the map when the Draw tool is minimized.
  • Measurement: Measure an area or distance in various units and/or click on the map to see map coordinates for the specified location.
  • Print: Print the current map view. Use drop down menus to choose page layout and print output file type. NOTE - it is typical for the print output result to take a couple minutes to generate .
  • Editor: Add new GIS features to the map, delete existing features and/or modify attributes for an editable feature. All edits are committed to GIS database upon saving.
  • Google Street View: Click the "Activate with map click" button and then click on the map to see Google Street View. NOTE - street view is not available in all areas.
