Barr Web Map Example
Mobile Device Help
  • Tools - To access map tools on a Smart Phone, tap the small arrow located on the left edge of the map screen. Tap the arrow again to close the tool menu.
    • Layers: Control map layer visibility and displays map symbols/legend. Check or un-check boxes to toggle layers on or off. Tap + sign to see individual layers and legends.
    • Bookmarks: Create bookmarks to save a defined map extent by tapping the "Add Bookmark" text. This will add a bookmark at the map's current extent. User bookmarks will persist until the web browser cache is cleared.
    • Print: Print the current map view. Use drop down menus to choose page layout and print output file type. NOTE - it is typical for the print output result to take a couple minutes to generate .
    • Google Street View: Click the "Activate with map click" button and then click on the map to see Google Street View. NOTE - street view is not available in all areas.
    • Editor: Add new GIS features to the map, delete existing features and/or modify attributes for an editable feature. All edits are committed to GIS database upon saving.

      • The web maps also allows multiple users to add and mark-up data stored in a secured relational database located on Barr's GIS server. In this environment, multiple users can edit the same features at the same time. Follow the steps outlined below to edit data in the Barr Web Map Example application:

      1. Tap on the "Editor" heading in the left pane
      2. Tap the "Start Editing" button
      3. Select the type of Comment from the menu and then click or tap on the map where at the location where the comment should get placed
      4. Populate the pop-up attribute window (note: an attachment can be added to the comment by browsing to the file)
      5. Use tools located on the bottom of the Editing tools as needed
      6. Tap the "Stop Editing" button when edits are complete
